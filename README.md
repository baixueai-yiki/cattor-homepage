




# 项目目标
开发一个简单的前后端网站，用于学习完整的 Web 开发流程。
项目应尽量覆盖真实网站开发中的基础流程，包括：
前端页面开发（HTML、CSS、JavaScript）
HTTP 请求与响应
Node.js 后端开发
数据持久化
Linux 服务器部署
Git 版本管理
域名与 HTTPS 配置

后续可逐步扩展：

用户登录
后台管理
Markdown 博客
图片上传
评论系统
WebSocket 实时聊天

## 设计思路
第一阶段先完成一个简单的留言板。
任何人都可以输入：
昵称
留言内容
点击提交后：
服务器保存留言。
刷新页面后：
所有留言仍然存在。
整个项目尽量不依赖大型框架，优先理解浏览器、服务器与数据之间的关系。
